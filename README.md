# movie-recommendation-system
A machine learning-based movie recommendation system using collaborative and content-based filtering.

This is  a content-based movie recommendation system using genre-vector , feature engineering and cosine similarity, modelling 100 films across 20 unique genres (Action, Sci-Fi, Film-Noir, War, etc.).

Designs a user profile builder that maps genre preferences to numerical vectors and ranks films by similarity score, returning personalised top-N recommendations.
